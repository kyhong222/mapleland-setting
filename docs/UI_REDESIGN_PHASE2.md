# UI 개편 Phase 2

## 목표
Phase 1에서 진행한 UI 개편을 기반으로 추가적인 레이아웃 및 정보 표시 개선

## 상단 UI 개선 (플레이어 vs 몬스터)

### 레이아웃
- 상단 UI의 너비를 하단 UI 너비와 일치시키기
- 플레이어와 몬스터 정보를 더 명확하게 구분

### 플레이어 정보 (좌측)
- **레벨만 표기** (기존에 표시되던 스탯, 물공/마공 제거)
- 향후 확장: 캐릭터 장비에 따른 이미지 렌더링 추가 예정

### 몬스터 정보 (우측)
공간이 넓어졌으므로 더 많은 정보 표기:
- HP (체력)
- 물리방어력
- 마법방어력
- 회피율
- 명중률

## 하단 UI 재구성 (4열 레이아웃)

### 새로운 4열 구조
1. **1열: 장비 + 인벤토리**
   - 장비 슬롯
   - 인벤토리 아이템 목록

2. **2열: 스탯 + 상세스탯**
   - 기본 스탯 (STR, DEX, INT, LUK)
   - 상세 스탯 정보

3. **3열: 버프**
   - 활성화된 버프 목록

4. **4열: 계산기**
   - **상단: 데미지 계산기** (우선순위)
   - **하단: 피격/회피 계산기**

### 배치 우선순위
- 데미지 계산기를 피격/회피 계산기보다 위쪽에 배치
- 사용자가 가장 자주 확인하는 정보를 상단에 위치

## 구현 방향

### 컴포넌트 구조
- 상단 UI와 하단 UI 간 일관된 너비 유지
- Grid 또는 Flexbox를 활용한 4열 레이아웃
- 각 열의 내용물에 따라 적절한 너비 비율 조정

### 스타일링
- 각 섹션 간 명확한 구분선
- 일관된 padding과 margin
- 반응형 고려 (필요시)
